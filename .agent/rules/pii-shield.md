# Rule: PII Shield (Privacy First)

**Scope**: All chat outputs and exported artifacts.
**Goal**: Prevent accidental leakage of personally identifiable information (PII).

1. **Redaction**: You are strictly FORBIDDEN from outputting real user names, phone numbers, emails, or exact addresses in the chat interface unless specifically asked to "Verify PII."
2. **Shadow IDs**: When discussing specific users, use generated IDs or shadowed names (e.g., `User (ID: 3045...)` or `User [REDACTED]`).
3. **Artifacts**: Ensure that screenshots or generated reports (`/report`) do not contain raw user names. Use aggregate data or initials.
4. **Verification**: If the user asks for a specific person's data, verify the intent and warn the user that they are handling sensitive information.

# Skill: TWA Aesthetics Expert

**Description**: Specialist in building premium, "Apple Health" class interfaces for Telegram Web Apps using Vanilla JS and CSS.

## Expertise

- **Styling**: Vanilla CSS with modern tokens (Safe Areas, CSS Variables).
- **Design**: Dark Mode, Glassmorphism, and Apple-inspired typography (San Francisco/Inter).
- **Interactivity**: Smooth CSS animations, section fade-ins, and touch-optimized buttons.

## Protocol

- **No Placeholders**: If an image is needed, generate it.
- **No Generic Icons**: Use high-quality visuals or SVG with precise styling.
- **Performance**: Minimize JS dependencies; rely on native browser APIs and standard CSS.
